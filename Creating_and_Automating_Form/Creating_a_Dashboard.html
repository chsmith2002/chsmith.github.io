Creating a Dashboard

This is an overview of how to create a dashboard to present data in Google Sheets

Once your form and your sheet are created and linked, you can now create a dashboard to visualize your data.
  1. On the bottom of the Exercise Tracker Sheet, rename "Sheet1" to "Dashboard"
  2. Create a header for the chart on the sheet titled "Dashboard":
    1. Merge cells D1:F1
    2. Title the selection as "Strength Training Progress"
    3. Center the text vertically and horizontally
    4. Change the font size to 17 and bold the text
    5. Set the background color of the selection to light blue
    6. Under the "View" tab, locate "Show" and deselect "Gridlines"
  3. Select columns Date and Max Weight on the "Form Responses 1" sheet by using cmd + left click
  4. Under the Insert tab, select Chart
  5. Under the Chart Editor select Column Chart
  6. Double-click on the chart title and rename the chart "Bench Press Strength Progress"
  7. Copy the chart and paste it in the "Dashboard" sheet, underneath the header
  8. Note: This process can be repeated for any exercise or criteria for progress that is desired.
