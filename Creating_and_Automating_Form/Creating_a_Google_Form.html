Creating a Google Form

In order to create a place to input data, a Google Form is required. The questions answered will automatically link with the Google Sheet, where the inputs can be organized and formatted to be shown on a Dashboard

Once inside of the blank Google Sheet, follow the instructions below to access and create the Form. For this particular instruction, the example will follow how to structure the form for tracking the max weight of a Barbell Bench PressThe exercise can be changed according to the user's intended use.
  1. On a blank Google Sheet, rename it "Exercise Tracker Data"
  2. Locate the "Create a New Form" Button under the Tools tab of your Google Sheet
  3. Change the title of the new form to "Exercise Tracker"
  4. Note: For more information on how to navigate Google Forms, see About Google Forms

  5. Rename "Untitled Question" to "Date"
  6. To the right of the name, choose "Date" for the question type
  7. Add a new question and title it "Exercise Type"
  8. To the right of the name, choose "Dropdown" for the question type
  9. Replace Option 1 with Bench Press
  10. Add a new question
    1. Title the question "Sets"
    2. Choose the "Short Answer" question type
    3. Select "Data Validation" under the "More Options" tab
    4. Set the input to "Number"
    5. Set the "Number" criteria to "Is number"
    6. Note: For the custom error text, any error message can be applied. A common error response in this case would be "Please enter number."

  11. Duplicate the questioned titled "Sets"
    1. Title the duplicate as "Repetitions"
  12. Duplicate the questioned titled "Repetitions"
    1. Title the duplicate as "Max Weight"
  13. Note: For this instruction, only Max Weight of the entire exercise will be tracked. To create a tracker for each individual weight used for each Set, simply repeat this process for any amount of Sets, while distinguishing which weight was used for each Set.
