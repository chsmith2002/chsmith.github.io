Tracking Workouts With Google Apps

An overview of Tracking Workouts using Google Sheets and Google Forms

Using Google Forms, an entry table can be created to input workout information. By creating specific questions with response criteria, users can track their workout information. The responses can then automatically populate into Google Sheets. The information entered into the Form can be formatted accordingly within the Sheet. By creating these documents, users are able to set criteria and track progress made while exercising. The data can be shown as raw figures or can be used to generate charts to visualize progress.

  • Creating a Google Form
    In order to create a place to input data, a Google Form is required. The questions answered will automatically link with the Google Sheet, where the inputs can be organized and formatted to be shown on a Dashboard
  • Linking a Form to Sheets
    Linking a form to Google Sheets allows for the data to be automatically updated in the spreadsheet whenever a new response is recorded.
